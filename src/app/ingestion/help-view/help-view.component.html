Ingestion Workflow Version

With this editor you can handle ingestion workflows. In this short help you can get information in the following sections:

  • Create a new ingestion workflow
  • View existing ingestion workflows
  • View statistics to existing ingestion workflow
  • Edit an ingestion workflow

View existing ingestion workflows

Here you can do following steps with the existing ingestion workflows:

  • Edit - Change the ingestion workflow
  • Delete - Remove the specific ingestion workflow
  • Statistics - See insertion statistics about the specific ingestion workflow

Edit an ingestion workflow

The editor of an ingestion workflow ist divided in different sections (which are presented in a menu on the left side):

  • Source Description - in this section the output source is described
  • CSV Parameters - if the template is for a CSV ingestion workflow here the structure of the CSV is described
  • SOS inputs - inputs which describes the inputs in the SOS
  • Mapping Outputs - Inputs - the mapping between the defined outputs and the inputs
